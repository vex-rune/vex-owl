# Web 前端（Vue3）
## 功能描述
系统PC端用户界面，提供完整的智能对话交互功能，是用户操作的主要入口
## 核心功能
- 🔐 **登录注册**
  - 邮箱密码登录
  - 新用户邮箱注册
  - 忘记密码重置
- 💬 **对话交互**
  - 流式对话打字效果展示
  - 支持PDF、图片文件上传
  - 会话列表管理（新建/删除）
  - 历史消息查看
- ⚙️ **模型配置**
  - 多AI平台自由切换
  - 自定义API Key加密配置
  - 默认对话模型偏好设置
- 📚 **记忆管理**
  - 查看所有自动提取的记忆
  - 手动新增/编辑/删除记忆
  - 记忆分类标签管理
- 🎨 **界面特性**
  - 响应式布局，适配不同屏幕尺寸
  - 深色/浅色主题切换
  - 对话内容一键复制、导出
## 技术栈
- Vue 3 + Vite
- Element Plus 组件库
- Axios 网络请求库
- SSE 客户端（流式响应接收）
## 开发环境
- Node.js >= 16.0.0
- 默认启动端口：5173
## 常用命令
```bash
# 安装依赖
npm install
# 启动开发环境
npm run dev
# 生产环境构建
npm run build
```
