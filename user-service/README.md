# User Service 用户服务
## 功能描述
用户账号和个性化配置管理服务，负责所有用户相关的业务逻辑处理
## 核心功能
- 👤 **用户账号管理**
  - 邮箱+密码注册功能
  - 邮箱+密码登录验证登录功能
  - 用户信息修改、密码重置
- ⚙️ **AI模型配置管理**
  - 保存用户自定义的AI平台配置
  - 支持多平台API Key加密存储
  - 用户默认对话模型偏好设置
  - 模型配置的增删改查
- 🔑 **API Key安全管理**
  - 敏感信息加密存储
  - 访问权限控制
## 技术栈
- Java + Spring Boot
- MyBatis Plus
- PostgreSQL 数据库
- BCrypt 密码加密
## 服务端口
- 默认启动端口：8081
