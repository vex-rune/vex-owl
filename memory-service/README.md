# Memory Service 记忆服务
## 功能描述
智能记忆管理服务，负责用户长期记忆的提取、存储和使用，让AI对话更有上下文连贯性
## 核心功能
- 🧠 **自动记忆提取**
  - 对话过程中自动识别关键信息
  - 智能提取用户偏好、重要事实
  - 自动去重和优化记忆内容
- 📝 **手动记忆管理**
  - 手动新增记忆条目
  - 记忆内容修改、删除
  - 记忆分类标签管理
- 🔍 **记忆检索**
  - 对话时自动语义匹配相关记忆
  - 智能召回相关上下文信息
  - 记忆相关性排序优化
- 🔄 **自动带入对话**
  - 对话时自动将相关记忆注入上下文
  - 提升多轮对话的连贯性和准确性
  - 记忆使用频率智能调整
## 技术栈
- Java + Spring Boot
- 向量数据库/全文检索
- 语义相似度匹配算法
## 服务端口
- 默认启动端口：8083
