# Vex-Owl 智能对话系统

## 📖 项目简介
Vex-Owl 是一个功能完整的多平台智能对话系统，支持多AI模型接入、长时记忆、文件解析等功能，提供Web端和移动端双端支持，采用微服务架构设计，易于扩展和部署。

## ✨ 功能特性
- 🎯 **多模型支持**：对接豆包、DeepSeek等主流AI平台，支持灵活切换
- 💬 **流式对话**：实现打字机效果的流式响应，提升对话体验
- 📚 **长时记忆**：自动提取并存储用户记忆，对话时自动带入上下文
- 📄 **文件解析**：支持PDF、图片等文件上传和内容解析
- 📱 **多端同步**：Web端(Vue3)和移动端(Flutter)功能1:1完全同步
- 🔐 **安全可靠**：JWT鉴权、网关统一入口，保障系统安全

## 🏗️ 技术架构
### 后端服务（4个Java微服务）
| 服务名称 | 核心功能 |
|---------|---------|
| **gateway 网关服务** | 统一入口、路由转发、JWT登录鉴权、跨域处理、文件上传 |
| **user-service 用户服务** | 邮箱+密码注册登录、用户模型配置管理（平台、apiKey、模型名） |
| **chat-service 对话服务（核心）** | 流式对话输出、多AI平台对接、PDF/图片解析、会话管理（新建/删除） |
| **memory-service 记忆服务** | 自动提取用户记忆、手动增删改查记忆、对话时自动带入记忆 |

### 前端工程
| 终端类型 | 技术栈 | 核心功能 |
|---------|-------|---------|
| **App 前端** | Flutter | 与Web端功能完全一致、移动端适配、本地缓存 |
