# 通用查询规范

## 适用范围
Vex-Owl项目所有分页多条件查询接口



## 1. 业务层规则

### 1.1 管理器类规范
| 项目 | 规范 |
|-----|------|
| 注入 | `private final EntityManager entityManager` |
| 方法权限 | public |
| 方法名 | query |
| 入参 | QueriesPageRequest |
| 返回值 | List<实体类> |
| 注解 | @Component + @RequiredArgsConstructor |

### 1.2 查询实现
- 通过JpaQueriesExecutor指定实体类和EntityManager
- 调用page方法完成分页查询



## 2. 控制层规则

### 2.1 接口规范
| 项目 | 规范 |
|-----|------|
| 请求方式 | POST |
| 路径 | /query |
| 入参 | @RequestBody QueriesPageRequest |
| 返回 | ApiResponse.success(xxxManager.query(request)) |

### 2.2 代码固定不变
- 通用查询接口代码固定，无需额外编写业务逻辑
- 直接调用业务层query方法
- 结果封装进ApiResponse



## 3. 数据类型规范

### 3.1 统一类型
- 所有ID字段统一使用String类型
- 新增、修改接口必须添加@Valid



## 4. 代码规范（纯示例）

### 4.1 业务层实现

```java
package com.vex.owl.user.app.manager;

import java.util.List;

import javax.persistence.EntityManager;

import org.springframework.stereotype.Component;

import com.vex.common.query.JpaQueriesExecutor;
import com.vex.common.query.QueriesPageRequest;
import com.vex.owl.user.domain.subject.entity.SubjectEntity;

import lombok.RequiredArgsConstructor;

/**
 * 用户模块通用查询实现
 */
@Component
@RequiredArgsConstructor
public class SubjectManager {

    private final EntityManager entityManager;

    /**
     * 用户-通用分页查询
     * <p>JPA通用分页+多条件动态查询</p>
     */
    public List<SubjectEntity> query(QueriesPageRequest queriesPageRequest) {
        return JpaQueriesExecutor.of(SubjectEntity.class, entityManager)
                .page(queriesPageRequest);
    }
}
```

### 4.2 控制层实现

```java
/**
 * 用户-通用查询
 * <p>分页+多条件组合查询</p>
 */
@PostMapping("/query")
public ApiResponse<SubjectEntity> query(@RequestBody QueriesPageRequest request) {
    return ApiResponse.success(subjectManager.query(request));
}
```

package com.vex.owl.user.api;

import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestBody;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

import com.vex.common.query.QueriesPageRequest;
import com.vex.owl.user.app.manager.SubjectManager;
import com.vex.owl.user.domain.subject.entity.SubjectEntity;

import lombok.RequiredArgsConstructor;

/**
 * 用户模块
 * <p>用户相关业务接口</p>
 */
@RestController
@RequestMapping("/api/v1/users")
@RequiredArgsConstructor
public class UserApi {

    private final SubjectManager subjectManager;

    /**
     * 用户-通用查询
     * <p>分页+多条件组合查询</p>
     */
    @PostMapping("/query")
    public ApiResponse<SubjectEntity> query(@RequestBody QueriesPageRequest request) {
        return ApiResponse.success(subjectManager.query(request));
    }
    
    // ... 其他接口
}
