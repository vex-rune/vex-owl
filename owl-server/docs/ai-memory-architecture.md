### 3.2 压缩算法详情

```
┌─────────────────────────────────────────────────────────────────┐
│                      CompressionEngine                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │                 1. 摘要压缩 (Summarize)                   │  │
│   ├─────────────────────────────────────────────────────────┤  │
│   │                                                                 │  │
│   │   输入：对话历史 [msg1, msg2, ..., msgN]                     │  │
│   │          │                                                   │  │
│   │          ▼                                                   │  │
│   │   处理：提取关键信息                                          │  │
│   │          • 关键决策                                           │  │
│   │          • 重要结论                                           │  │
│   │          • 用户偏好                                           │  │
│   │          • 工具结果摘要                                        │  │
│   │          │                                                   │  │
│   │          ▼                                                   │  │
│   │   输出：压缩后的摘要                                          │  │
│   │          [summary: {key_points, decisions, preferences}]     │  │
│   │                                                                 │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │                 2. 删除压缩 (Prune)                       │  │
│   ├─────────────────────────────────────────────────────────┤  │
│   │                                                                 │  │
│   │   删除条件：                                                   │  │
│   │          • 重复内容 (similarity > threshold)                 │  │
│   │          • 低重要性 (importance < min_threshold)             │  │
│   │          • 时间久远 (age > max_age)                          │  │
│   │          • 噪音数据 (noise_score > threshold)                  │  │
│   │                                                                 │  │
│   │   保留策略：                                                   │  │
│   │          • 系统提示                                           │  │
│   │          • 最新消息 (last N)                                 │  │
│   │          • 高相关性 (relevance > threshold)                   │  │
│   │          • 工具调用结果                                         │  │
│   │                                                                 │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │                 3. 重排压缩 (Reorder)                      │  │
│   ├─────────────────────────────────────────────────────────┤  │
│   │                                                                 │  │
│   │   排序维度：                                                   │  │
│   │          • 相关性 (与当前任务的语义相似度)                    │  │
│   │          • 重要性 (importance score)                         │  │
│   │          • 时序性 (时间顺序)                                 │  │
│   │          • 依赖性 (消息间的依赖关系)                          │  │
│   │                                                                 │  │
│   │   权重配置：                                                   │  │
│   │          • relevance_weight    │ 0.4                      │  │
│   │          • importance_weight   │ 0.3                      │  │
│   │          • recency_weight      │ 0.2                      │  │
│   │          • dependency_weight   │ 0.1                      │  │
│   │                                                                 │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
