# Chat Service 对话服务（核心服务）
## 功能描述
系统核心业务服务，负责所有AI对话相关的逻辑处理，是整个系统的核心能力中心
## 核心功能
- 💬 **流式对话输出**
  - 实现打字机效果的流式响应
  - SSE长连接实时推送
  - 支持对话内容分段输出
- 🤖 **多AI平台对接**
  - 兼容豆包、DeepSeek、OpenAI等主流AI平台
  - 支持动态切换对话模型
  - 统一的平台调用适配层
- 📄 **文件解析能力**
  - PDF文件上传与内容解析
  - 图片OCR识别与内容提取
  - 支持多格式文件内容理解
- 📑 **会话管理**
  - 新建会话、删除会话
  - 历史消息持久化存储
  - 会话列表查询与管理
- 🧠 **上下文管理**
  - 多轮对话上下文自动维护
  - 对话历史自动截断优化
## 技术栈
- Java + Spring Boot
- 各大AI平台SDK
- Apache PDFBox（PDF解析）
- Tesseract OCR（图片识别）
- SSE 流式输出
## 服务端口
- 默认启动端口：8082
