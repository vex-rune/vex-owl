# Java 日志规范

## 适用范围
所有Java模块、所有Java开发人员



## 1. 日志框架

- 日志门面：SLF4J
- 日志实现：Logback（Spring Boot默认）
- 注解支持：Lombok @Slf4j



## 2. 日志级别使用（文字规格）

### 2.1 级别定义

| 级别 | 使用场景 | 示例 |
|-----|---------|------|
| **TRACE** | 最详细的追踪信息 | 方法进入/退出、循环次数 |
| **DEBUG** | 开发调试信息 | SQL语句、变量值、条件分支 |
| **INFO** | 正常业务流程 | 接口调用、任务开始/完成、状态变化 |
| **WARN** | 潜在问题 | 参数校验失败、缺少某些数据、性能警告 |
| **ERROR** | 错误异常 | 方法失败、异常捕获、外部调用失败 |

### 2.2 选择原则
- 生产环境关闭DEBUG/TRACE
- 正常业务流程用INFO
- 异常但不影响运行时用WARN
- 需要用户关注的问题用ERROR



## 3. 敏感信息保护（文字规格）

### 3.1 绝对禁止日志输出
- 用户密码
- 支付密码
- 密钥、Token、SecretKey
- 手机号（可部分脱敏后输出）
- 身份证号
- 银行卡号
- 完整地址（可脱敏后输出）

### 3.2 脱敏处理原则
- 手机号：138****5678
- 身份证：110101****1234****
- 银行卡：****1234（只留后4位）



## 4. 日志内容规范（文字规格）

### 4.1 必要日志
- 服务启动/停止
- 接口入口（参数脱敏）
- 接口出口（返回值脱敏）
- 业务关键节点
- 异常发生位置（包含堆栈）

### 4.2 日志内容要求
- 使用占位符而非字符串拼接
- 包含上下文信息（订单号、用户ID等）
- 日志消息要清晰表达业务含义
- 避免日志过长

### 5.2 参数校验日志

```java
package com.vex.owl.common.validator;

import lombok.extern.slf4j.Slf4j;

@Slf4j
public class Validator {
    
    public static void validate(UserRequest request) {
        if (request.getUsername() == null) {
            log.warn("用户名不能为空");
            throw new ValidationException("用户名不能为空");
        }
        if (request.getUsername().length() < 3) {
            log.warn("用户名长度不足, length={}", request.getUsername().length());
            throw new ValidationException("用户名长度不足");
        }
    }
}
```

### 5.3 性能日志

```java
package com.vex.owl.report.app;

import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Service;

@Slf4j
@Service
public class ReportService {
    
    public Report generateReport(String type) {
        long startTime = System.currentTimeMillis();
        log.info("开始生成报表, type={}", type);
        
        try {
            Report report = doGenerate(type);
            
            long cost = System.currentTimeMillis() - startTime;
            log.info("报表生成完成, type={}, cost={}ms", type, cost);
            
            if (cost > 5000) {
                log.warn("报表生成耗时较长, type={}, cost={}ms", type, cost);
            }
            
            return report;
        } catch (Exception e) {
            long cost = System.currentTimeMillis() - startTime;
            log.error("报表生成失败, type={}, cost={}ms", type, cost, e);
            throw e;
        }
    }
}
```

### 5.4 禁止示例

```java
// ❌ 禁止：日志输出密码
log.info("用户登录, username={}, password={}", username, password);

// ❌ 禁止：日志输出完整手机号
log.info("发送验证码, phone={}", phone);

// ❌ 禁止：字符串拼接
log.info("用户信息: " + user.toString());

// ❌ 禁止：日志输出token
log.error("用户登录成功, token={}", token);

// ❌ 正确：使用占位符
log.info("用户登录, username={}", username);
```

### 5.5 脱敏日志示例

```java
package com.vex.common.utils;

public class LogUtils {
    
    public static String maskPhone(String phone) {
        if (phone == null || phone.length() < 7) {
            return "****";
        }
        return phone.substring(0, 3) + "****" + phone.substring(phone.length() - 4);
    }
    
    public static String maskIdCard(String idCard) {
        if (idCard == null || idCard.length() < 8) {
            return "****";
        }
        return idCard.substring(0, 4) + "********" + idCard.substring(idCard.length() - 4);
    }
}

// 使用
log.info("发送验证码, phone={}", LogUtils.maskPhone(phone));
```
