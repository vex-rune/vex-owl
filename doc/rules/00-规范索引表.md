## 四、使用说明

### 4.1 按场景加载
在 .trae/rules 配置文件中使用 scene 标签指定适用场景：

```yaml
# 项目规则
alwaysApply: true
scene: project_structure

# Java开发
alwaysApply: true
scene: java_ddd

# API开发
alwaysApply: true
scene: rest_api
```

### 4.2 规范层级
1. **通用基础规范**：所有项目、所有语言必须遵守
2. **语言专项规范**：特定语言必须遵守
3. **整体架构规范**：项目整体架构必须遵守
4. **接口编写规范**：API开发必须遵守

### 4.3 规范优先级
通用基础规范 > 语言专项规范 > 整体架构规范 > 接口编写规范
