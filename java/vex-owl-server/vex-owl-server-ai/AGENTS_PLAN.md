# Vex-Owl AI Agent 编排系统规划

> **项目目标**：构建一个类 Trae 的智能对话系统，集成豆包大模型 + Agent 编排能力
> **技术栈**：Java 21 | Spring AI | PostgreSQL (pgvector) | WebFlux
> **架构模式**：DDD 领域驱动设计

---

## 📋 总体路线图

```
阶段一：基础设施搭建（第 1-2 周）
阶段二：核心对话功能（第 3-4 周）
阶段三：Agent 编排框架（第 5-7 周）
阶段四：高级功能增强（第 8-10 周）
阶段五：生产化优化（第 11-12 周）
```

---
